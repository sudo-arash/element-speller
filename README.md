# 🧪 Element Speller

Spell words using chemical element symbols from the periodic table.

## Features

- Backtracking algorithm for the best decomposition
- Fast memoized search
- Interactive Vue UI
- Built with TypeScript

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- Vue 3
- TypeScript
- Tailwind CSS
- Vite
